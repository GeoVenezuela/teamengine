The teamengine-spi Module

This module provides a simple extensibility framework and a REST API for test execution. A test runner that implements the TestRunExecutor interface is invoked by a controller (TestSuiteController) that is discovered using the JDK ServiceLoader mechanism.

Support is provided for executing test suites using the TestNG framework. The REST API is based on JAX-RS 1.1.
